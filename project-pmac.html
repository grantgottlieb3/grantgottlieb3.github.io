Permanent Magnet AC Motor Drive

Simulink model of a PMAC drive using sine-triangle PWM with third-harmonic injection; step torque commands, closed-loop current control, and DC-link dynamics. :contentReference[oaicite:15]{index=15}

Read Report (PDF) See Plots

Model Summary

The system couples a DC source + LC filter, three-phase inverter, PWM modulator with 3rd-harmonic injection, and PMAC machine with dq-current control. A MATLAB script generates optimal Iqs, Ids lookup tables versus torque. During tests at 500 rpm, torque is stepped from 0 → 400 → −400 Nm to examine steady/transient response, currents, voltages, and battery/DC link behavior. :contentReference[oaicite:16]{index=16}

Selected Results

Phase Quantities

120°-shifted phase voltages and currents; current responds smoothly due to inductance while voltage steps with modulation. :contentReference[oaicite:17]{index=17}

DC Side

Filtered DC current tracks battery current; example values ~55 A at 400 Nm and −48 A at −400 Nm. :contentReference[oaicite:18]{index=18}

Torque Tracking
